## 📦 Product Description

### What is it?

LokalBoost Google Maps Scraper is a professional desktop application that lets you **automatically extract bulk business data** from Google Maps. Business name, address, phone number, website, rating, review count, category and more — exported to CSV or JSON in seconds.

### Who is it for?

- **Digital Marketing Agencies** — Build prospect lists at scale
- **B2B Sales Teams** — Automate lead generation
- **SEO Specialists** — Gather competitor data for local SEO audits
- **Data Analysts** — Create regional business intelligence reports
- **Entrepreneurs** — Research your target market
- **Researchers** — Compile industry-specific datasets

### 🔍 Powerful Search
- **Multiple Keywords** — Search multiple industries simultaneously
- **Country, State, City/District Targeting** — 250 countries, thousands of cities
- **Smart Location System** — Jitter coordinates to cover every corner

### ⚡ High Performance
- **Parallel Scraping** — Run 10+ queries concurrently
- **Pagination Support** — Multiple pages per query
- **Smart Rate Control** — Built-in rate limiting with anti-detection

### 🛡️ Anti-Detection Technology
- **20+ User-Agent Rotation** — Each request mimics a different device
- **Pseudo-Cookie Generation** — Realistic browser fingerprint simulation
- **Auto Retry & Exponential Backoff** — Smart recovery from 429/403 errors

### 🌐 Proxy Support
- **Rotating Proxy** — Single gateway URL with unlimited IP rotation
- **Sticky Proxy** — Bulk proxy list with persistent session support
- **Built-in Proxy Test** — One-click connection test after setup
- **Compatible with** DataImpulse, Bright Data, Smartproxy, Oxylabs & all HTTP proxy providers

### 📊 Advanced Data Management
- **Real-time Results Table** — Data appears instantly as it's scraped
- **Inline Search Filter** — Instant search by name, address, phone
- **City/District Dropdown** — Auto-parsed from address data
- **Phone/Website Toggle** — Show only businesses with contact info
- **Column Sorting** — Sort by name, rating, reviews (ascending/descending)
- **Stats Bar** — Total results, average rating, phone/website percentages

### 💾 Flexible Export
- **CSV & JSON** formats
- **Column Selection** — Export only the fields you need
- **Filtered or All Data** — Export based on active filters or everything
- **Custom File Names**

### 🔄 Auto-Save
- **Per District** — Auto-save when district changes
- **Per State** — Auto-save when state changes
- **Per Result Count** — Save every N results (e.g., every 200)

## 📖 How to Use

### 1️⃣ Installation
- Download the `.dmg` (Mac) or `.exe` (Windows) installer
- Follow the setup wizard
- Launch the application

### 2️⃣ Login
- Sign in with your license key
- New users: Click "Register" to create an account

### 3️⃣ Configure Your Search
- Enter **keywords** (one per line)
  - Example: `restaurant`, `hair salon`, `law firm`
- **Select location** — Country → State → City/District
- Adjust **scraping depth** (page count, concurrent queries)

### 4️⃣ Settings (Optional)
- Click ⚙️ Settings to configure proxy, auto-save, and data filters
- If using a proxy, click **Test Proxy** to verify the connection

### 5️⃣ Start Scraping
- Click **Start Scraping**
- Monitor the real-time progress bar and live statistics
- Results appear in the table instantly

### 6️⃣ Filter & Export
- Use table filters to narrow down results
- Click 📥 **Export** to download as CSV or JSON

## ❓ FAQ

**Q: Is this software legal?**
A: The software collects publicly available data from Google Maps. Personal use is generally acceptable. For commercial use, please verify your local regulations.

**Q: Do I need to use a proxy?**
A: No. It works without a proxy. However, for large-scale scraping (10,000+ results), we recommend using a proxy to prevent IP blocking.

**Q: Which proxy providers are compatible?**
A: DataImpulse, Bright Data, Smartproxy, Oxylabs, and any provider that offers HTTP proxy support.

**Q: How many results can I extract?**
A: There's no technical limit. It's possible to extract 100,000+ results in a single session. The auto-save feature ensures your data is safely stored during large operations.

**Q: What data fields are included?**
A: Business name, address, phone number, rating (1-5), review count, category, website URL, Google Maps link, search query.

**Q: Will I receive updates?**
A: Yes. All updates are free as long as your license is active.
